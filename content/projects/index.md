# Projects

A collection of technical projects I've built, ranging from cloud-native applications to mobile apps with machine learning integration.

## Featured Projects

### [[projects/terrasense|TerraSense]]
**Geospatial Intelligence Platform**

Built at AI ATL 2025 Hackathon, TerraSense processes NASA satellite data into actionable insights. Users can analyze vegetation health using NDVI metrics with 25 years of historical data and AI-powered predictions.

**Tech Stack:** React.js, CesiumJS, FastAPI, MongoDB Atlas, Google Earth Engine API, Google Gemini API

---

### [[projects/teachify|Teachify]]
**Cloud-Native Student Learning Hub**

Developed during my internship at Nexus Technologies Group, Teachify leverages AWS services and SageMaker Foundation models to transform unstructured notes into structured, interactive study content.

**Tech Stack:** AWS (SageMaker, Lambda, Amplify), Cloud Architecture, Foundation Models

---

### [[projects/resumi|Resumi]]
**Student Portfolio & Academic Tracking App**

A mobile application for students to create portfolios of extracurricular activities and track academic performance (weighted and unweighted GPA) with personalized ML insights.

**Tech Stack:** React Native, GraphQL API, AWS Amplify, AWS Cognito, AWS AppSync, Machine Learning (k-means clustering)

---

### [[projects/jira-automation|JIRA Time Tracking Automation]]
**Automated Employee Hour Tracking System**

Developed an automated script utilizing the Atlassian JIRA API and Activity Timeline API to gather employee hours and send project hour logging reminders.

**Tech Stack:** Python, Atlassian JIRA API, AWS Lambda, Serverless Architecture
